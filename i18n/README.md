# 6章 vue-i18nのLazy loadingとvue-router

vue-i18nのLazy loadingの実装をしている章のサンプルコードです。
またLazy loadingする際にvue-routerのナビゲーションガードが深く関わったのでその実装もあります。

## commands

### yarn

#### インストール

```
$ yarn
```

#### 動作確認

```
$ yarn serve
```

### npm

#### インストール

```
$ npm install
```

#### 動作確認

```
$ npm run serve
```
