# 4章 Vuexのtips

Vuexを使ったモーダルの表示のサンプルコードです。

## デモ

[https://mya-ake.github.io/vue-tips-sample-demos/samples/modal_store/](https://mya-ake.github.io/vue-tips-sample-demos/samples/modal_store/)

## commands

### yarn

#### インストール

```
$ yarn
```

#### 動作確認

```
$ yarn serve
```

### npm

#### インストール

```
$ npm install
```

#### 動作確認

```
$ npm run serve
```
